# 项目说明
这是一个简易的NSFW的web api服务器，可以检测图片是否含有NSFW内容。基于[opennsfw2](https://github.com/bhky/opennsfw2)和[fastapi](https://fastapi.tiangolo.com/)。

# 使用说明

## 使用docker（推荐）
`docker run --name nsfw-server -p 8000:8000 -d allyxmiko/nsfw-server:latest`
